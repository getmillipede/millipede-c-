# millipede-csharp

[![Build Status](https://secure.travis-ci.org/getmillipede/millipede-csharp.svg)](https://travis-ci.org/getmillipede/millipede-csharp)

Print a beautiful millipede

## Installation

### compile & try

Using Windows

````sh
c:\> csc millipede.cs
c:\> millipede.exe 42
```

Using Mono
````sh
$ mcs millipede.cs
$ mono millipede.exe 42
```
